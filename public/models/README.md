# Free, High-Quality GLB Avatars for MindSafe

Download and add these GLB files to the public/models/ folder for a diverse avatar gallery.

## Recommended Sources

- [Mixamo Characters (Adobe)](https://www.mixamo.com/#/?page=1&type=Character) — Free, requires Adobe account. Download as GLB.
- [Ready Player Me Sample Avatars](https://github.com/readyplayerme/avatars) — Diverse, CC0 license.
- [Sketchfab - Free Human 3D Models](https://sketchfab.com/tags/human?sort_by=-likeCount&features=downloadable&license=cc0) — Filter for CC0, GLB format.
- [Clara.io - Free Characters](https://clara.io/library?query=human&public=true&gameCheck=true) — Download as GLB.
- [Kenney.nl - Human Characters](https://kenney.nl/assets/human-characters) — Stylized, but diverse and free.

## Instructions

1. Download GLB files from the links above.
2. Place them in the `public/models/` folder.
3. Name files descriptively (e.g., african_male.glb, asian_female.glb, etc.).
4. Restart your development server if running.

---

**Tip:** For maximum diversity, download at least 5-10 avatars covering different races, genders, and features.
